## üßÆ Formula Used
> **Total Cost = Base Charge + (Weight √ó Distance √ó Rate Per Km)**  
> Base charge = ‚Çπ50  
> Rate per km per kg = ‚Çπ0.05  

Estimated delivery days = `distance / 500`

---

## üß† Example
If a package weighs **5 kg** and travels **1000 km**,  
Total = 50 + (5 √ó 1000 √ó 0.05) = ‚Çπ300  
Delivery = 2 days (approx.)
